# 可视化ETL设计器

基于 React Flow 的可视化ETL流程设计器。

## 功能特性

### 基础功能
- ✅ **拖拽式设计** - 从侧边栏拖拽节点到画布，支持视觉反馈
- ✅ **节点连接** - 通过连接点连接节点，构建数据流，带连接验证
- ✅ **节点配置** - 点击节点进行详细配置
- ✅ **保存/加载** - 保存ETL流程到本地存储
- ✅ **导入/导出** - 支持JSON格式导入导出
- ✅ **多种节点类型** - 数据源、转换、目标、流程控制

### 增强功能
- ✅ **键盘快捷键** - 支持常用快捷键操作
  - `Ctrl/Cmd + C` - 复制节点
  - `Ctrl/Cmd + V` - 粘贴节点
  - `Ctrl/Cmd + X` - 剪切节点
  - `Ctrl/Cmd + Z` - 撤销
  - `Ctrl/Cmd + Shift + Z` / `Ctrl/Cmd + Y` - 重做
  - `Ctrl/Cmd + A` - 全选节点
  - `Ctrl/Cmd + F` - 搜索节点
  - `Delete` / `Backspace` - 删除选中节点
  - `ESC` - 关闭搜索和右键菜单

- ✅ **多选和批量操作** - 支持框选和批量操作多个节点
  - 批量复制/删除
  - 批量对齐

- ✅ **节点对齐** - 支持6种对齐方式
  - 左对齐、居中、右对齐
  - 顶部对齐、垂直居中、底部对齐

- ✅ **右键菜单** - 丰富的上下文菜单
  - 节点右键菜单：配置、复制、剪切、删除、复制节点
  - 画布右键菜单：粘贴、全选

- ✅ **节点搜索** - 快速定位节点
  - 按名称或类型搜索
  - 自动高亮匹配节点
  - 自动定位到第一个匹配节点

- ✅ **改进的画布体验**
  - 拖拽时视觉反馈（画布高亮）
  - 节点悬停效果
  - 改进的小地图（彩色节点标识）
  - 网格对齐（20px网格）
  - 平滑动画连接

- ✅ **撤销/重做** - 支持50步操作历史

## 使用方法

1. **添加节点**：从左侧面板拖拽节点到画布
2. **连接节点**：从一个节点的输出点拖拽到另一个节点的输入点
3. **配置节点**：点击节点，然后点击"配置"按钮打开配置面板
4. **保存流程**：点击"保存"按钮保存当前流程
5. **加载流程**：点击"加载"按钮加载之前保存的流程
6. **清空画布**：点击"清空"按钮清除所有节点和连接

## 快捷键

### 键盘快捷键
- `Ctrl/Cmd + C` - 复制选中节点
- `Ctrl/Cmd + V` - 粘贴节点
- `Ctrl/Cmd + X` - 剪切选中节点
- `Ctrl/Cmd + Z` - 撤销操作
- `Ctrl/Cmd + Shift + Z` 或 `Ctrl/Cmd + Y` - 重做操作
- `Ctrl/Cmd + A` - 全选所有节点
- `Ctrl/Cmd + F` - 打开搜索
- `Delete` / `Backspace` - 删除选中节点
- `ESC` - 关闭搜索和右键菜单

### 鼠标操作
- **鼠标滚轮** - 缩放画布
- **拖拽空白区域** - 平移画布
- **拖拽节点模板** - 从侧边栏拖拽到画布创建节点
- **连接节点** - 从一个节点的输出点拖拽到另一个节点的输入点
- **单击节点** - 选中节点
- **框选** - 拖拽选择多个节点（按住 Ctrl/Cmd）
- **右键节点** - 显示节点上下文菜单
- **右键画布** - 显示画布上下文菜单
