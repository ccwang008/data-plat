# 可视化ETL设计器

基于 React Flow 的可视化ETL流程设计器。

## 功能特性

- ✅ **拖拽式设计** - 从侧边栏拖拽节点到画布
- ✅ **节点连接** - 通过连接点连接节点，构建数据流
- ✅ **节点配置** - 点击节点进行详细配置
- ✅ **保存/加载** - 保存ETL流程到本地存储
- ✅ **多种节点类型** - 数据源、转换、目标、流程控制

## 节点类型

### 数据源节点
- MySQL数据源
- PostgreSQL数据源
- 文件数据源

### 转换节点
- 字段映射
- 数据清洗
- 数据过滤
- 数据聚合

### 目标节点
- 数据库目标
- 文件目标
- API目标

### 流程控制节点
- 条件分支
- 循环
- 并行

## 使用方法

1. **添加节点**：从左侧面板拖拽节点到画布
2. **连接节点**：从一个节点的输出点拖拽到另一个节点的输入点
3. **配置节点**：点击节点，然后点击"配置"按钮打开配置面板
4. **保存流程**：点击"保存"按钮保存当前流程
5. **加载流程**：点击"加载"按钮加载之前保存的流程
6. **清空画布**：点击"清空"按钮清除所有节点和连接

## 快捷键

- 鼠标滚轮：缩放画布
- 拖拽空白区域：平移画布
- 双击节点：快速配置

## 技术实现

- React Flow - 流程图核心库
- Ant Design - UI组件库
- TypeScript - 类型安全
